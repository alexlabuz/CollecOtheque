<h1>CollecOthèque</h1>
<p>CollecOthèque est une application permettant de ranger vos livres dans une bibliothèque virtuelle.<br> 
Projet SLAM 4 pour le lycée Guy Mollet.</p>

<h2>Fonctionnalités de l'application</h2>

<h3>Fonctionnalités principales</h3>
<ul>
	<li>Création de bibliothèque ✅</li>
	<li>Ajout d'étagère aux bibliothèques existantes ✅</li>
	<li>Ajout de collections aux étagères ✅</li>
	<li>Ajout de livre aux collections, via leurs code ISBN ✅</li>
	<li>Retrait de livre de la bibliothèque ✅</li>
	<li>Consultation des livres d'une étagère ou d'une collection ✅</li>
	<li>Consultation des détails d'un livre de la bibliothèque ✅</li>
	<li>Transfére d'un livre d'une bibliothèque, à une autre ✅</li>
	<li>Suppression de collection, étagère ou bibliothèque ✅</li>
</ul>

<h3>Fonctionnalités bonus</h3>
<ul>
	<li>Utilisation de la caméra pour scanner les codes barres (ISBN) ✅</li>
	<li>Sauvegarde des données dans le cloud (API Personnalisé) ❌ <i>(Non disponible)</i></li>
</ul>
<p><b>Info importante : </b>Si lors de l'ouverture du scanner l'écran reste noir, tapez plusieurs fois sur l'écran activer le scanner.</p>

<h2>Librairie utilisées</h2>
<ul>
	<li>Android RecyclerView</li>
	<li>Google Material design</li>
	<li>Android Volley</li>
	<li>Android Room</li>
	<li>Budiyev Code Scanner</li>
</ul>
